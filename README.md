## Cara Berkontribusi

1. Clone repository ini
2. Buat branch baru untuk fitur/tugas Anda
3. Commit perubahan Anda
4. Push ke repository
5. Buat Pull Request untuk review

## Alur Kerja Git

```bash
# Clone repository
git clone https://github.com/Khanz69/SewaKu.git

# Buat branch baru
git checkout -b nama-fitur-anda

# Commit perubahan
git add .
git commit -m "Deskripsi perubahan"

# Push ke repository
git push origin nama-fitur-anda
```
