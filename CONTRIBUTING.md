# Panduan Kontribusi

Terima kasih atas kontribusi Anda pada proyek SewaKu!

## Alur Kerja

1. **Buat Branch Baru**
   - Gunakan format: `fitur/nama-fitur` atau `perbaikan/nama-perbaikan`
   - Contoh: `fitur/halaman-login`, `perbaikan/bug-form`

2. **Commit Message**
   - Tulis pesan commit yang jelas dan deskriptif
   - Gunakan bahasa Indonesia untuk konsistensi

3. **Pull Request**
   - Buat Pull Request setelah selesai mengerjakan fitur
   - Minta review dari anggota tim lain
   - Pastikan tidak ada konflik dengan branch utama

## Aturan Tim

- Komunikasikan perubahan besar sebelum mulai mengerjakan
- Review kode rekan tim dengan baik dan berikan feedback konstruktif
- Jaga kualitas kode dan dokumentasi
